### Step 1: Push Your Repository
```bash

git remote add origin https://github.com/YOUR_USERNAME/flappy-kiro-cloud-platform.git
git push -u origin main
```

### Step 2: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** tab
3. Scroll down to **Pages** section
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch
6. Choose **/ (root)** folder
7. Click **Save**

### Step 3: Configure Custom Path (Optional)
If you want the demo at `/docs/` instead of root:
1. In Pages settings, choose **/docs** folder instead of **/ (root)**
2. Your demo will be at: `https://yourusername.github.io/flappy-kiro-cloud-platform/`
